Gameplay:
Destroy the enemy turrets before they destroy you. 

Controls:
W - Forward
A - Left
S - Backward
D - Right
Mouse - Aim
Left Click - Fire
P - Pause

Level Select:
Selecting a level designates it as the loaded level, you still need to select "Play Loaded Level" after selecting a level.

Important Note:
After you beat a level, it will return you to the main menu with the next level preloaded.
If you lose the level, it will return you to the main menu with the same level reloaded.
